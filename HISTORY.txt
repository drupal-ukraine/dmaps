WHAT THIS MODULE FOR?
---------------------
This is repository for development a new Drupal 8 module which include modules from drupal 7:
 - Static Map: https://www.drupal.org/project/staticmap
 - Location: https://www.drupal.org/project/location
 - GMap Module: https://www.drupal.org/project/gmap

PROJECT ON Drupal.org
----------------------
https://www.drupal.org/project/dmaps

ITERATIONS OF DEVELOPMENT:
----------------------
1. Port module Location module to Drupal 8
2. Port submodules of location module to Drupal 8
3. Port and union GMAP module to Drupal 8
4. Port submodules of GMAP module to Drupal 8
5. Port formatters from staticmap module to Drupal 8.
6. Release first stable version.
7. Add feature improvements.
8. Release second stable version.
